# 🕒 Sistema de Folha de Ponto

## 📌 Sobre o projeto

O **Sistema de Folha de Ponto** é uma aplicação web desenvolvida com **React e TypeScript**, com o objetivo de registrar e acompanhar a jornada de trabalho de usuários.

A aplicação permite o controle de horários de entrada, saída e pausas, além de exibir o histórico de registros de forma simples e organizada.

---

## 🚀 Funcionalidades

* ✔️ Registro de entrada e saída
* ✔️ Controle de pausas
* ✔️ Visualização de histórico de pontos
* ✔️ Cálculo básico de horas trabalhadas
* ✔️ Interface simples e intuitiva

---

## 🛠️ Tecnologias utilizadas

* React
* TypeScript

---

## 🧠 Regras de funcionamento

* O usuário registra os horários manualmente
* O sistema evita registros duplicados consecutivos
* O cálculo de horas é feito com base nos registros do dia

## ⚙️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/Gremoryxzz/FolhaPonto.git
```

2. Acesse a pasta do projeto:

```bash
cd FolhaPonto
```

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npm start
```
